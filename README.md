# projketinr4
# Menaxhimi i Kodit Burimor dhe Bashkëpunimi në GitHub  
## Projekt në C++

## Përshkrimi i Projektit
Ky projekt është realizuar në gjuhën programuese **C++** dhe ka për qëllim
demonstrimin praktik të përdorimit të **GitHub si sistem për menaxhimin e kodit
burimor dhe bashkëpunimin në ekip**. Gjatë zhvillimit të projektit janë aplikuar
praktikat standarde të version control, duke përfshirë përdorimin e
repositories të përbashkëta, branch-eve për zhvillim paralel, commits të
strukturuara dhe pull requests për integrimin e kodit.

Projekti synon të tregojë aftësinë e anëtarëve të ekipit për të bashkëpunuar
në mënyrë efektive, për të menaxhuar ndryshimet në kod dhe për të ruajtur një
historik të qartë dhe të organizuar të zhvillimit të projektit.

## Struktura e Projektit

## Bashkëpunimi dhe Menaxhimi i Versioneve
- Zhvillimi i projektit realizohet përmes **branch-eve** për punë paralele
- Çdo ndryshim në kod regjistrohet me **commits të qarta dhe domethënëse**
- Integrimi i ndryshimeve në degën kryesore (**main**) bëhet përmes
  **Pull Requests**
- Pull Requests përdoren për **code review, diskutim dhe miratim të kodit**
- Sinkronizimi i punës realizohet përmes operacioneve **push** dhe **pull**

## Ekzekutimi i Projektit
Kodi burimor i projektit mund të shkarkohet drejtpërdrejt nga
GitHub repository. Pas shkarkimit, projekti mund të kompilohet dhe
ekzekutohet në çdo mjedis që mbështet gjuhën C++.
